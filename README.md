# Contact Manager

A command-line Contact Manager built with Python that allows users to store and manage contacts efficiently.

## Features

- Add new contacts
- View all contacts
- Search contacts by name
- Edit existing contacts
- Delete contacts
- Automatic save/load using JSON
- Input validation

## Technologies Used

- Python
- JSON
- File Handling

## Future Improvements

- Phone number validation
- Email validation
- Sort contacts alphabetically
- Export contacts to CSV
- GUI version using Tkinter
